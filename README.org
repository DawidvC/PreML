* PreML: A preprocessor for SML

Supports =do=-syntax, and more.

Put this in your =.emacs=:
#+BEGIN_SRC: lisp
(setq sml-use-command
      (concat
       "local "
       "val filei = \"%s\" "
       "val fileo = filei ^ \".preml\" "
       "val _ = OS.Process.system (\"preml \\\"\" ^ filei ^ \"\\\"\") "
       "val _ = use fileo "
       "val _ = OS.FileSys.remove fileo "
       "in end")
      )
#+END_SRC
